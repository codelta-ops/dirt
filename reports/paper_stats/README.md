# Paper Stats

This folder stores paper-ready data exports for the DIRT+ manuscript.

Files:
- `table_3_1_dataset_statistics.csv`: dataset statistics used in the paper.
- `table_3_2_training_config.csv`: current training configuration from `full_model`.
- `table_3_3_main_results.csv`: main results table. Baseline rows can come from the manuscript, DIRT+ rows come from current logs.
- `table_3_4_ablation.csv`: ablation results aggregated from `DIRT_3` and `DIRT_4` test metrics.
- `table_3_5_stage_comparison.csv`: stage1/stage2 comparison for `DIRT+_3` and `DIRT+_4`, with validation and test metrics kept in separate columns.
- `paper_stats.md`: markdown preview of all exported tables.
- `paper_stats.json`: machine-readable bundle of all exported tables.
